# 各国KYC要求展示平台

一个简洁的信息展示平台，专门用于展示欧美日韩等发达国家和小地区的KYC（了解您的客户）验证要求。

## ✨ 特点

- 📊 展示11个主要发达国家的KYC要求
- 🔍 支持按地区筛选（北美、欧洲、亚洲、大洋洲）
- 🔎 快速搜索功能
- 💡 清晰显示每个国家所需文件和验证要求
- ⏱️ 显示处理时间和费用信息
- 🎯 直观的KYC级别分类

## 🚀 快速开始

### 前提条件

安装Node.js（推荐v18+）

### 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

## 🎨 功能说明

### 首页
- 展示所有支持的国家
- 点击任意国家卡片查看详细信息
- 支持按地区筛选
- 支持搜索国家名称

### 详情页
显示每个国家的：
- 📋 所需文件清单
- ✅ 验证要求
- ⏱️ 处理时间
- 💰 费用信息
- 🏷️ KYC级别

这是一个纯展示平台，不包含任何交易功能或用户认证功能。专注于为需要了解不同国家KYC要求的人员提供清晰、易于查找的信息。
